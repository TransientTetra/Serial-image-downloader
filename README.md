# Serial-image-downloader
A program to download images from a website with regular numeric subpages, ie. xkcd.com

# List of supported websites #
(this is not an exhaustive list, just those that are confirmed)  
-xkcd.com  
-kuvaton.com  
-drmcninja.com  
-gunnerkrigg.com  
-hiagb.com  
-gunshowcomic.com
